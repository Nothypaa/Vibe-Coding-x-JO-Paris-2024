# Jeux Olympiques Paris 2024 - Webapp Interactive

Webapp interactive realisee dans le cadre du **Projet supervise B2 - Session 2 : Vibe Coding x JO Paris 2024**.

## Apercu

Cette webapp transpose les 4 exercices de la Session 1 (ecosysteme JO Paris 2024) en une application web fonctionnelle et interactive, developpee en Vibe Coding (developpement assiste par IA).

## Sections

### 1. Infographie de l'Ecosysteme JO
- Cartes interactives avec flip 3D (cliquer pour reveler les details)
- Chiffres cles animes (35 sites, 329 epreuves, 206 nations, 45 000 benevoles)
- Organigramme de l'organisation (CIO, COJO, SOLIDEO, Etat)
- Graphique de repartition du budget (8,8 Mds EUR)

### 2. Dashboard Performances Athlete
- Profil complet de Leon Marchand (4 Or, 1 Bronze)
- Tableau de resultats filtrable (Individuel / Relais) et triable par colonnes
- Modales de details pour chaque epreuve
- 4 graphiques interactifs : temps par epreuve, repartition medailles, evolution records, tableau des nations

### 3. Heritage Olympique
- Carte interactive SVG des sites olympiques (Paris & Seine-Saint-Denis)
- Scroll storytelling en 5 chapitres (Urbanisme, Environnement, Mobilite, Social, Bilan)
- Comparaison Avant/Apres avec donnees chiffrees
- Temoignages d'habitants (Marie, Ahmed, Fatima)
- Statistiques animees (3 000 emplois, 45 000 benevoles, 400 clubs, 1M billets solidaires)
- Analyse equilibree : points positifs et points de vigilance

### 4. Innovations Technologiques
- 8 fiches d'innovation filtrables par categorie (Diffusion, Securite, Sport, Ecologie)
- Jauges d'impact animees au scroll
- Graphique radar d'impact par domaine
- Comparaison technologique avec les editions precedentes

## Technologies

- **HTML5** - Structure semantique
- **CSS3** - Design responsive, animations, 3D transforms, CSS Grid/Flexbox
- **JavaScript** (Vanilla) - Interactivite, filtres, tri, modales, scroll animations
- **Chart.js** - Graphiques interactifs (bar, doughnut, line, radar)
- **Google Fonts** - Poppins

## Fonctionnalites

- Design responsive (desktop, tablette, mobile)
- Navigation fixe avec surlignage de section active
- Ecran de chargement anime
- Barre de progression de scroll
- Compteurs animes au scroll
- Cartes 3D avec flip au clic
- Filtres par categorie et type
- Tableau triable par colonnes
- Modales de detail
- Carte interactive SVG avec tooltips
- Particules animees en arriere-plan (hero)
- Bouton retour en haut

## Lancer en local

Ouvrir `index.html` dans un navigateur. Aucune installation requise.
